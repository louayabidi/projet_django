<p><span style="color: rgb(45, 49, 66);">Ce projet est une application web développée avec le framework Django permettant la gestion complète d’une bibliothèque numérique. Les utilisateurs peuvent consulter la liste des livres disponibles, ajouter de nouveaux ouvrages, visualiser les détails de chaque livre, et télécharger les fichiers associés. L’application repose sur une base de données PostgreSQL pour le stockage des informations, et intègre une interface conviviale facilitant la navigation et la recherche. Ce projet illustre une architecture moderne basée sur le modèle MVC, combinant efficacité, sécurité et extensibilité.</span></p>
